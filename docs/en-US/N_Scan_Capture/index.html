Nouveautés Open Bee Scan Capture

Function of archiving and exporting data over a given period

This feature now allows you to archive and export the data stored in the database over a period to lighten it.

Ability to drag and drop a document on the banner of the console

It is now possible to drag and drop a document on the banner of the console in order to process it as a priority without passing through the source directory of images to import.

Preview of previous and next invoices

A preview of the previous and next invoices is now displayed in the video coding screen.

Third party repository accessible from the video coding screen

The third-party repository is now accessible from the videcoding screen. This makes it possible to no longer exit this screen to view / modify this repository.

Addition of a calculation rule to constrain the search or validation of an index

The new engine of Open Bee Scan Capture now allows you to add customizable calculation rules to introduce a search or validation constraint for an index.

Example : Calculation rule. Incl tax = excl tax + VAT ammount

Addition of a zone rule to constrain search for an index

The new engine of Open Bee Scan Capture now allows to add parametric rules allowing to introduce a constraint of search or validation of an index.

Text search functionality on an invoice

The keyboard shortcut CTRL + F allows you to search for text on an invoice either from the video coding screen, or from the table displaying invoices waiting for validation (in the "Videocoding" tab).

Functionality to copy and paste regex and markers

A new copy / paste functionality of regex and markers allows copying and / or pasting all regex or markers of an index.

Addition of default Value on Indexes

It is now possible to set a default value for an index either for all folders or for all suppliers in the same folder or a specific supplier.

Line article capture functionality: item/article code, quantity, unit price, etc …

The new engine of Open Bee Scan Capture is now able to capture the line artciles of an invoice (set of indexes belonging to the same line: item code, quantity, unit price, sub-amount, ...)

Scripted index (Index that can take any manual value or calculated according to the values of other indexes)

It is now possible to configure an index that can take any manual or calculated value based on the values of other indexes using a JavaScript programming language.

Mutualization of third parties

A third party already entered in a folder of Open Bee Scan Capture can now be reused in another folder of the application. An automatic proposal of shared third parties is also put in place to facilitate the choice of this one to the user. This makes it possible to centralize third-party template changes between different folders, thus avoiding repetition of changes for each shared third party.

Ability to absorb documents from a folder Open Bee™ Portal

It is now possible to absorb documents from an Open Bee™ Portal folder. This absorption can be done automatically or manually

Accounting

Division of a line entry over a period

It is possible to divide an amount by period to obtain sub-amounts per month.

Ability to zip the accounting export with PDF files

It is now possible when exporting a batch of entries possible to zip a folder containing the accounting entries/records file with the PDF files of the documents associated with the entries.

Ability to perform scripts in the entries labels

Entries labels can be built simply based on variable or can result from a javascript script.

Generation of an entry line based on the value of an index

It is now possible to generate an entry line or analytical breakdown at 100% automatically based on the values of an index.
