Nouveautés Open Bee Scan Capture

Fonction d'archivage et d'exportation des données sur une période donnée

Cette fonctionnalité permet désormais d'archiver et d'exporter les données stockées dans la base de données sur une période afin d'alléger celle ci.

Possibilité de glisser & déposer un document sur la bannière de la console

Il est désormais possible de glisser & déposer un document sur la bannière de la console afin de le traiter en priorité sans passer par le répertoire source des images à importer.

Prévisualisation des factures précédente & suivante

Une prévisualisation des factures précédente & suivante est désormais affichée dans l'écran de vidéocodage.

Référentiel tiers accessible depuis l'écran de vidéocodage

Le référentiel tiers est désormais accessible depuis l'écran de vidécodage. Cela permet de ne plus sortir de cet écran pour consulter / modifier ce référentiel.

Ajout de règle de calcul permettant de contraindre la recherche ou la validation d'un index

Le nouveau moteur de Open Bee Scan Capture permet désormais d'ajouter des règles paramétrables de calcul permettant d'introduire une contrainte de recherche ou de validation d'un index.

Exemple : Règle de calcul. TTC = HT + Montant TVA

Ajout de règle de zone permettant de contraindre la recherche d'un index

Le nouveau moteur de Open Bee Scan Capture permet désormais d'ajouter des règles paramétrables permettant d'introduire une contrainte de recherche ou de validation d'un index.

Fonctionnalité de recherche de texte sur une facture

Le raccourci clavier CTRL + F permet de rechercher du texte sur une facture soit depuis l'écran de vidéocodage, soit depuis le tableau affichant les factures en attente de validation (dans l'onglet "Videocodage").

Fonctionnalité de copier/coller de regex et de marqueurs

Une nouvelle fonctionnalité de copier/coller de regex et de marqueurs permet de copier et/ou de coller l'ensemble des regex ou des marqueurs d'un index.

Ajout de valeur par défaut sur les Index

Il est désormais possible de définir une valeur par défaut pour un index soit pour l'ensemble des dossiers ou pour l'ensemble des fournisseurs du même dossier ou un fournisseur bien précis.

Fonctionnalité de capture de lignes articles : code article, quantité, prix unitaire, etc…

Le nouveau moteur de Open Bee Scan Capture est désormais capable de capturer les lignes articles d'une facture (ensemble d'index appartenant à la même ligne : code article, quantité, prix unitaire, sous montant,...)

Index scripté (Index pouvant prendre n'importe quelle valeur manuelle ou calculée en fonction des valeurs des autres index)

Il est désormais possible de configurer un index pouvant prendre n'importe quelle valeur manuelle ou calculée en fonction des valeurs des autres index à l'aide d'un langage de programmation en JavaScript.

Mutualisation des tiers

Un tiers déjà renseigné dans un dossier de Open Bee Scan Capture peut être désormais réutilisé dans un autre dossier de l'application. Une proposition automatique des tiers partagés est aussi mise en place pour faciliter le choix de celui-ci à l'utilisateur. Cela permet de centraliser les modifications du modèle tiers entre les différents dossiers, évitant ainsi la répétition des modifications pour chaque tiers partagé.

Possibilité d'absorber des documents depuis un dossier Open Bee™ Portal

Il est désormais possible d'absorber des documents depuis un dossier Open Bee™ Portal. Cette absorption peut se faire de manière automatique ou manuelle

Comptabilité

Division d'une ligne d'écriture sur une période

Il est possible de diviser un montant en fonction d'une période pour obtenir des sous-montants par mois.

Possibilité de zipper l'export comptable avec les fichiers PDF

Il est désormais possible lors de l'export d'un lot d'écrture possible de zipper un dossier contenant le fichier des écritures comptables avec les fichiers PDF des pièces associés aux écritures.

Possibilité d'effectuer du script dans les libéllés d'écritures

Les libellés d'écritures peuvent être construits simplement à base de variable ou peuvent résulter d'un script javascript.

Génération d'une ligne d'écriture en fonction de la valeur d'un index

Il est désormais possible de générer une ligne d'écriture ou une ventilation analytique à 100% automatiquement en fonction des valeurs d'un index.
