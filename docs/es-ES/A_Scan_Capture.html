Ameliorations Open Bee Scan Capture

Restriction des documents traités aux extensions PDF / TIFF / TIF / PNG / DOC / DOCX

Seules les extensions PDF / TIFF / TIF / PNG / DOC / DOCX sont autorisées à être traitées par Open Bee Scan Capture. Toutes autres extensions sont automatiquement déplacées dans le dossier des documents en erreur.

Possibilité d'utiliser la date de traitement du document comme métadonnée envoyée à Open Bee™ Portal

La date de traitement du document par Open Bee Scan Capture peut désormais être utilisée comme métadonnée envoyée à Open Bee™ Portal.

Suppression d'un regex depuis l'interface de personnalisation

Il est désormais possible de supprimer une regex depuis l'interface de personnalisation (depuis l'écran de vidéocodage).

Affichage du nom de l'index sur les zones personnalisées

Le nom de l'index est désormais affiché sur les zones personnalisées pour une meilleur identification.

Pouvoir mettre une valeur de liste par défaut sur un index

Il est désormais possible de définir une valeur par défaut pour les index de type liste.

Option de remplacement des caractères interdits pour les sorties (Open Bee™ Portal, Windows)

Il est désormais possible de remplacer automatiquement des caractères interdits pour les sorties (Open Bee™ Portal, Windows).

Mise à jour de Mongo DB de 3.2 à 3.4

La base de données Mongo DB a été mise à jour de 3.2 à 3.4

Possibilité de changer les regex des index de type date

Il est désormais possible de changer les regex des index de type date.

Amélioration du moteur de mots clés pour les mots clés contenant des espaces

Le moteur de mots clés permettant d'identifier un fournisseur à l'aide de mot clé a été amélioré pour les mots clés contenant des espaces

Possibilité d'accéder au paramétrage des index directement depuis l'interface de vidéocodage

Il est possible désormais d'accéder au paramétrage des index directement depuis l'interface de vidéocodage en double cliquant sur le nom de l'index.

Amélioration du système d'absorption des fichiers

Le système d'absorption des fichiers a été amélioré pour éviter des erreurs dans le cas où les fichiers sont utilisés par un autre processus. Les fichiers sont désormais déplacés dans un sous dossier nommé "Fichiers en cours".

Règle de validation

Une règle de validation peut être ajoutée sur un index afin que l'utilisateur soit averti depuis l'interface de vidéocodage lorsque la valeur de celui-ci n'est pas correcte.

Utilisation de regex dans les conditions de transcodage d'un index

Il est désormais possible d'utiliser des regex dans les conditions de transcodage d'un index.

Avertissement de l'utilisateur si une regex n'est pas syntaxiquement correcte

Un avertissement visuel est affiché dans l'interface de configuration des regex si la syntaxe d'une regex saisi par l'utilisateur n'est pas correcte.

Possibilité de réinitialiser complètement un modèle tiers

La fonctionnalité permettant de réinitialiser les personnalisations d'un modèle a été remplacée par une réinitialisation totale de celui-ci pour repartir du modèle par défaut.

Ajout d'un indicateur de chargement lors de la modification d'un index

Il est désormais possible de suivre l'avancement de la modification d'un index sur tous les modèles contenus dans Open Bee Scan Capture.

Optimisation de la réinterprétation d'un document

Si le modèle n'a pas subi de changement, le document n'est plus réinterprété inutilement

Ajout d'une interface permettant de gérer les différents modèles de documents dans Open Bee Scan Capture

Une nouvelle interface permet de gérer les différents modèles de document.

Optimisation des performances globales de la console Open Bee Scan Capture

Les performances d'affichage de la console ont été améliorées, ce qui permet un chargement plus rapide des informations notamment dans le cas d'un grande volumétrie de document.

Ajout d'un champ séparateur pour les nombres décimaux

Possibilité de forcer le séparateur qui doit être présent sur les documents pour considérer le texte en valeur décimale.

Bloquage de la validation si un document est détecté en doublon

Il est désormais possible de configurer Open Bee Scan Capture pour bloquer la validation des documents détectés en tant que doublon.

Possibilité de choisir le nombre de décimales voulu

Il est désormais possible de choisir le nombre de décimales à afficher dans Open Bee Scan Capture.

Ce qui permet de gérer par exemple les devises : -sans décimales (Afrique et Pacifique) -avec 3 décimales (Tunisie)

Ajout d'une constante %OTHER% dans le tableau de transcodage

Il existe désormais deux constantes dans le tableau transcodage: %ALL% pour gérer le paramétrage comptable sur toutes les valeurs possibles et %OTHER% permettant de gérer le paramétrage sur toutes les lignes non paramétrées.

Comptabilité

Division d'un montant en pourcentage et enregistrement dans le modèle d'écriture

Il est désormais possible de diviser un montant en pourcentage dans le tableau d'écriture comptable et que cette division soit enregistrée dans le modèle d'écriture.

Possibilité de paramétrer l'écart analytique

L'interface de configuration comptable vous permet désormais de choisir si l'écart comptable doit être: refusé, autorisé, équilibré automatiquement

Importation de l'analytique en format fichier

Il est désormais possible d'importer un plan analytique à partir d'un format fichier.

Importation de compte tiers sans centralisateur

Il est désormais possible via l'import fichier d'importer des comptes tiers qui n'ont pas de centralisateur par défaut.

Fusion de deux ventilations automatiques sur des plans différents

Lorsque deux ventilations analytiques proviennent d'une génération automatique, celles-ci peuvent être fusionnées si elle appartiennent à deux plans différents.

Possibilité de fusionner les lignes d'écritures comptables générées automatiquement

Il est désormais possible de fusionner les lignes d'écritures générées automatiquement pour celles ayant des comptes identiques.

Amélioration de la recherche des sections analytique

Il est désormais possible de rechercher une section analytique par le début ou le milieu de son nom ou son code.

Blocage de la validation d'une écriture si un plan analytique obligatoire n'est pas rempli

Désormais, si un plan anlytique est obligatoire, la validation de la facture sera bloquée si ce plan n'a pas été ventillé.
